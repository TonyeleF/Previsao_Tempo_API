# 🌦️ App de Previsão do Tempo

Aplicação web que consome uma API de previsão do tempo para exibir informações em tempo real como temperatura, clima e condições atmosféricas de uma cidade.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API (consumo de API)
* Manipulação do DOM

## 📌 Funcionalidades

* 🌍 Busca de cidade pelo usuário
* 🌡️ Exibição da temperatura atual
* ☁️ Exibição das condições climáticas (nublado, chuva, sol, etc.)
* 📊 Informações adicionais (umidade, vento, etc)
* ⚠️ Tratamento de erro para cidade inválida
* 📱 Interface simples e responsiva

## 🌐 API utilizada

Este projeto utiliza uma API pública de previsão do tempo, como:

* OpenWeatherMap API

## 🔑 Configuração da API

Para rodar o projeto corretamente, é necessário:

1. Criar uma conta na API utilizada
2. Gerar uma chave (API Key)
3. Inserir a chave no código JavaScript

Exemplo:

```js id="j0k1l2"
const apiKey = "SUA_API_KEY";
```
